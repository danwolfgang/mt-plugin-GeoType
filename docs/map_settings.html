GoogleMap Settings

Settings are configurable under every blog's Plugin Settings. It controls default values and general restrictions on map appearances. The option to use additional fields for every map is available, as well as options regarding the RSS feeds for the blog.

Keep in mind that all of these settings are at the blog-level; that is, the settings of one blog will not affect those of another.
