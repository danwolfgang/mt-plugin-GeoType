Why can't I see a map...
  • in my Edit Entry screen?

    1. If you can't see any areas that would display GeoType data on your Edit Entry Screen and you see the message 'To Enable GeoType, add the appropriate library mapping keys in your settings', set up this blog's GoogleMaps API key.

    2. If you are able to see the areas where GeoType data should appear (such as the extra fields for the location's name and address) but the map itself is not displaying, the GoogleMaps API key for this blog may be incorrect for the Movable Type installation. Create a new system-wide key to fix it. Note: This should not affect the key used in any of your published pages.


What is the difference between the Promo Images (Thumbnail, Large Promo, Billboard Image or Flash, Optional Promo) and the photos I can see in the Entry Photo?
  • The "Promo Images" are the photos displayed on the Radar Homepage, on a blog's main page (like on Features), and in the search results page. The Entry Photos are the ones that are displayed within the built archive pages only.

Why can't I see a photo preview for a photo I know I've uploaded and spelled correctly?
  • Your photo must have been uploaded through the current blog in order to view the preview and have the MTPhoto tag make the correct image link. Try uploading the photo again to the blog you're working in.
  • If that doesn't work, you can always check if the photo was uploaded into your blog properly by checking through your internet browser. Go to the main page of your blog on the staging server (Features, Fresh Intelligence, etc) and type your file's name directly after the slash of the web address. If the picture is there, it should show only the image in the web browser without any fancy stylings. If the picture is missing, a webpage with text like "The requested page could not be found".

What if my photo doesn't have a caption?
  • Then your photo won't have the caption text displayed. The photo itself will still display on the page.

How can I tell if the photo I've uploaded is large or small?
  • If you've associated it with an entry, then in the entry's Photo Listing, the photo Style field (Adding Photos, Figure 4i)) should state if it's a small or large image. The cutoff to determine an image's sizing is also displayed on the entry's Photo Listing page at the top in red (Adding Photos, Figure 4c).
