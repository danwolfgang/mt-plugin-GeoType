Tags Documentation

Container Tags
Tag Description
< MTGeoTypeLocations /> A container tag that loops through all locations for a given context. The following contexts are checked:
* If there is an entry in context, it will loop through all entries for that location
* If there is no entry in context and the page is an archive, it will loop through all locations for entries in that archive
* If there is no entry or archive in context, it will check the stash
* If these checks fail, it will return empty
Conditional Tags
Tag Description
< MTGeoTypeIfLocation /> Checks for the presence of a location in context
< MTGeoTypeIfLocationExtended /> Checks for extended location values in context to test whether extended locations have been activated in the plugin settings
Template Tags
Tag Description
Base location values:
< MTGeoTypeLocationName > The name of the location
< MTGeoTypeLocationId > The ID of the GeoType::Location record
< MTGeoTypeLocationLatitude > Latitude of the location
< MTGeoTypeLocationLongitude > Longitude of the location
< MTGeoTypeCoords > Coordinates of the locations ("Long, Lati").
< MTGeoTypeLocation > Returns the location value; likely to be a street address ("1060 West Addison Street, Chicago, IL") that has been used for geocoding
< MTGeoTypeMap > Generates a Google map for a given location or locations using preferences set in plugin settings.
< MTGeoTypeHeader > Generates calls to include <script> and <style> items for proper creation and display of the GeoTypeMap tag's output. To be included in the document's <head>
Extended Location Values:
< MTGeoTypeLocationCrossStreet > Returns the cross street, if any, set in the GeoType::ExtendedLocation record.
< MTGeoTypeLocationDescription > Returns the description, if any, set in the GeoType::ExtendedLocation record.
< MTGeoTypeLocationHours > Returns the hours, if any, set in the GeoType::ExtendedLocation record. Likely to be a string.
< MTGeoTypeLocationPhone > Returns the phone number, if any, set in the GeoType::ExtendedLocation record.
< MTGeoTypeLocationPlaceId> Returns the PlaceID record, if any, set in the GeoType::ExtendedLocation record. See this page for details.
< MTGeoTypeLocationRating > Returns the rating, if any, set in the GeoType::ExtendedLocation record. A numeric value ("3.5").
< MTGeoTypeLocationThumbnail > Returns the thumbnail value, if any, set in the GeoType::ExtendedLocation record. A string ("http://www.example.com/visitor_thumb.png").
< MTGeoTypeLocationURL > Returns the URL, if any, set in the GeoType::ExtendedLocation record.
GeoRSS values:
<GeoRSS_Namespace > GeoRSS_Namespace
<GeoRSS_Entry > Generates <point> tag or <geo:lat>/<geo:long> tags for location, depending on style selected in plugin settings
<GeoRSS_Channel > Currently unused
