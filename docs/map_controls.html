GoogleMap Settings: Map Controls

Certain controls from the blog-level GeoType settings will hold for all GoogleMaps used in the blog; other settings are just default values. The following controls will hold for all maps:

  1. Map Width
  2. Map Height
  3. Zoom Controls
  4. Map Type (On/Off)
  5. Overview Control
  6. Scale Control

These are the modifiable default-valued controls:

  1. Default Zoom Level
  2. Default Map Type

It should be noted that all of these screenshots are previews of the GoogleMaps generated on the blog-level GeoType settings.

Width and Height
  • Control the width and height of all GoogleMaps in blog
  • In px
  • Generate actual-sized preview in GeoType settings and in Edit Entry screen
  • Should not be less than 200px by 200px to allow for room for GoogleMaps controls
