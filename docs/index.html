GeoType: A User's Guide

The GeoType plugin for Movable Type provides a simple interface to Google Maps on blog entries. It creates locations, which are pushpin points that appear on a single map per entry.

Definitions

  • location / map location- saved data that maps an address to
    a latitude and longitude that makes it possible to create a
    single thumbtack on a map generated by GoogleMaps
