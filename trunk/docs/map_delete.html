Removing Locations from an Entry

If you'd like to remove the location thumbtack without permanently deleting the location data, these instructions are for you. If the permanent removal of the location is what you need, visit permanently removing locations.


Figure 1
Click on a location's 'Remove' link to take it off
the entry's map.



Remove a Location from an Entry

Simply click the 'Remove' link in the location's row on the Edit Entry Screen. The location's thumbtack should instantly disappear from the entry map's preview. [ Figure 1 ]

Save the entry to save the location removal.
